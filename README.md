## 背景

想要设计个人`logo`，找了很多资料，发现

1. [logo.svg](https://github.com/bubkoo/logo.svg)可以自定义`logo`图案，并且能够生成`svg`文件
2. [svglib](https://github.com/deeplook/svglib)可以转换`svg`到`png`
  
写了这个脚本，方便大家使用
